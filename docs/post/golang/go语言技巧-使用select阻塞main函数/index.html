Go语言技巧-使用select{}阻塞main函数

2017-09-06

很多时候我们需要让main函数不退出，让它在后台一直执行，例如：

func main() {
	for i := 0; i < 20; i++ { //启动20个协程处理消息队列中的消息
		c := consumer.New()
		go c.Start()
	}
	select {} // 阻塞
}
